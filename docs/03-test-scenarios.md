# Тестовые сценарии telegram-mcp-plus

## 1. tg_me — Информация об аккаунте

| # | Сценарий | Вход | Ожидаемый результат |
|---|----------|------|---------------------|
| 1.1 | Успешный запрос | — | JSON с id, first_name, last_name, username, phone |
| 1.2 | Нет авторизации | Сессия отсутствует | Ошибка: "Not authorized. Run: telegram-mcp-plus auth" |
| 1.3 | Сессия истекла | Протухшая сессия | Ошибка с инструкцией переавторизации |

## 2. tg_dialogs — Список диалогов

| # | Сценарий | Вход | Ожидаемый результат |
|---|----------|------|---------------------|
| 2.1 | Все диалоги | — | Массив из ≤20 диалогов с name, type, title, last_message |
| 2.2 | Только непрочитанные | only_unread=true | Только диалоги с непрочитанными сообщениями |
| 2.3 | Пагинация | offset="..." | Следующая страница диалогов |
| 2.4 | Пустой список | Новый аккаунт без чатов | Пустой массив |
| 2.5 | Диалог с длинным сообщением | Сообщение > 500 символов | Текст обрезан до 200 символов + "..." |

## 3. tg_dialog — Сообщения из чата

| # | Сценарий | Вход | Ожидаемый результат |
|---|----------|------|---------------------|
| 3.1 | Получить сообщения | name="username" | Массив сообщений с id, text, date, sender |
| 3.2 | Пользовательский лимит | name="...", limit=5 | Ровно 5 сообщений |
| 3.3 | Пагинация по offset | offset_id=12345 | Сообщения старше указанного ID |
| 3.4 | Несуществующий чат | name="nonexistent_xyz" | Ошибка: "Dialog not found: nonexistent_xyz" |
| 3.5 | Медиа-сообщение | Сообщение с фото | text="[Photo]" или caption, без бинарных данных |
| 3.6 | Пустой чат | Чат без сообщений | Пустой массив |

## 4. tg_read — Пометить прочитанным

| # | Сценарий | Вход | Ожидаемый результат |
|---|----------|------|---------------------|
| 4.1 | Успешная пометка | name="username" | "Marked as read: username" |
| 4.2 | Уже прочитан | Чат без непрочитанных | Успех (идемпотентно) |
| 4.3 | Несуществующий чат | name="nonexistent" | Ошибка: "Dialog not found" |

## 5. tg_send — Отправка сообщения

| # | Сценарий | Вход | Ожидаемый результат |
|---|----------|------|---------------------|
| 5.1 | Отправка текста | name="username", text="Hello" | JSON с message_id, timestamp |
| 5.2 | Отправка себе | name="me", text="Note" | Сообщение в Saved Messages |
| 5.3 | Пустой текст | name="...", text="" | Ошибка: "Text cannot be empty" |
| 5.4 | Длинный текст | text длиной 5000 символов | Успешная отправка (Telegram лимит ~4096) |
| 5.5 | Несуществующий получатель | name="nonexistent" | Ошибка: "Dialog not found" |
| 5.6 | Канал без прав | Канал где нельзя писать | Ошибка: "Cannot send message to this chat" |

## 8. Авторизация (CLI)

| # | Сценарий | Вход | Ожидаемый результат |
|---|----------|------|---------------------|
| 8.1 | Успешная авторизация | Валидный телефон + код | "Authorization successful!" + сессия сохранена |
| 8.2 | С 2FA | Аккаунт с 2FA | Запрос пароля → успех |
| 8.3 | Неверный код | Неправильный код | Ошибка: "Invalid code" |
| 8.4 | Неверный телефон | Невалидный формат | Ошибка: "Invalid phone number" |
| 8.5 | Нет TG_APP_ID | env не задан | Ошибка: "TG_APP_ID is required" |
| 8.6 | Повторная авторизация | Сессия уже есть | Перезаписать сессию |
